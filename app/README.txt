Pour démarrer l'application WEB, vous devez :

Avoir Python (dans sa version récente)
Avoir le module Flask et SQL3
Avoir un navigateur WEB fiable

###################################################################

Comment télécharger Python ?

Rendez-vous sur https://www.python.org/ et télécharger la version la plus récente

Comment télécharger les modules Flask et SQL3 ?

Rendez-vous sur l'invite de commande de votre système et lancer la commande : python -m pip install Flask
Cela installera les fichiers nécessaires pour lancer l'application WEB

###################################################################

1 - Télécharger tout le contenu de l'application via le GITHUB
2 - Lancer le fichier projet.py dans le fichier principal
3 - Copier/coller la suite de chiffre que le programme vous donne dans votre navigateur WEB (ex : http://127.0.0.1:5000/)
4 - Vous arrivez normalement sur une page WEB, vous pouvez aller faire votre anecdote sur "Ajoutez une anecdote" pour ajouter votre propre anecdote
ou vous pouvez visualiser les anecdotes sur "Visualiser la base de donnée".

###################################################################

Comment marche l'application ?

L'application se forme sur une base de donnée qui receuille les différentes anecdotes depuis la page "Ajoutez", celle-ci les implémentes dans
l'application et sont visionables depuis "Visualiser la base de donnée", une utilisation simple et rapide.
